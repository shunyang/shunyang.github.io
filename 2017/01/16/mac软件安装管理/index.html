mac软件安装管理

发表于 2017-01-16 | 分类于 工具 |

brew使用：

brew主要用来下载一些不带界面的命令行下的工具和第三方库来进行二次开发

  • Homebrew的安装路径及文件夹
    Homebrew将本地的/usr/local初始化为git的工作树，并将目录所有者变更为当前所操作的用户，以后的操作将不需要sudo。

-bin 用于存放所安装程序的启动链接（相当于快捷方式）
-Cellar 所以brew安装的程序，都将以[程序名/版本号]存放于本目录下
-etc brew安装程序的配置文件默认存放路径
-Library Homebrew 系统自身文件夹
brew下载的软件的目录 ~/Library/Caches/Homebrew

  • 常用指令
    brew update 更新brew
    brew install XXX 安装xxx软件
    brew upgrade xxx 更新xxx软件
    brew cleanup 清理旧版本的包缓存
    brew list -version 查看你安装过的包列表（包括版本号）
    brew outdated 查看你的包是否需要更新
    brew –cache 查看通过brew下载软件的源文件

    brew cask使用

    brew cask主要用来下载一些带界面的应用软件，下载好后会自动安装，并能在mac中直接运行使用
    brew cask的目录为：/usr/local/Caskroom
    brew cask search 列出所有可以被安装的软件
    brew cask search drop 查找所有和drop相关的应用
    brew cask info thunder 查看 迅雷 应用的信息
    brew cask uninstall qq 卸载 QQ
    brew cask install google-chrome 安装谷歌浏览器
